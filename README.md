# 🐱 astrbot_plugin_doro_today

一个AstrBot插件，可以从Doro结局图片中随机抽取一张并发送给用户，同时@发送者。

## 🚀 功能介绍

- **随机抽取Doro结局图片**：从插件目录下的`doro`文件夹中随机抽取一张图片。
- **@发送者**：发送图片时自动@发送者。
- **使用**：发送指令`/dorotoday`或者`/今日doro`,`/doro结局`, `/doro今日`即可触发插件功能。
- 插件小版本更新（0.0.X）是添加图片，建议按需更新
